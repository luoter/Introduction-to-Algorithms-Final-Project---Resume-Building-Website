# 📤 如何分享给其他人

让其他人快速使用 AI 简历生成器的几种方法！

---

## 🎯 推荐方式（按简单程度排序）

### 方式一：打包分享（最推荐！）⭐

适合分享给不太懂技术的朋友：

1. **运行打包脚本**
   ```
   双击运行 「打包分享.bat」
   ```

2. **压缩分享**
   ```
   进入「打包输出」文件夹
   右键点击「AI简历生成器」文件夹
   选择「发送到 -> 压缩(zipped)文件夹」
   ```

3. **发送给朋友**
   ```
   通过微信、QQ、邮箱等发送压缩包
   ```

4. **朋友使用步骤**
   ```
   1. 解压压缩包
   2. 双击运行「快速启动.bat」
   3. 等待浏览器自动打开
   4. 开始使用！
   ```

---

### 方式二：直接部署到云服务

适合想要一个永久在线链接的用户：

#### 推荐服务（免费！）

1. **Vercel**（最简单）
   - 访问 https://vercel.com/
   - 注册账号
   - 导入项目
   - 点击部署

2. **Netlify**
   - 访问 https://www.netlify.com/
   - 注册账号
   - 导入项目
   - 配置：
     - 构建命令：`npm run build`
     - 发布目录：`dist`

3. **GitHub Pages**
   - 推送到 GitHub
   - 按 README 中的步骤配置

---

### 方式三：分享 dist 文件夹

如果你已经运行过 `npm run build`，可以直接分享 `dist` 文件夹：

1. **构建项目**
   ```bash
   npm run build
   ```

2. **分享 dist 文件夹**
   ```
   可以将 dist 文件夹放到：
   - 任何静态文件托管服务
   - 自己的服务器
   - GitHub Releases
   ```

3. **对方使用**
   ```
   如果放到服务器上，直接访问网址即可
   如果是本地文件，双击 dist/index.html 也能用
   ```

---

## 📋 完整的分享文件清单

当你分享项目时，确保包含以下文件：

```
AI简历生成器/
├── src/                    # 源代码文件夹
├── index.html             # 入口 HTML
├── package.json           # 项目配置
├── vite.config.js         # Vite 配置
├── README.md              # 完整文档
├── QUICKSTART.md          # 快速开始
├── 使用说明.txt            # 简单说明
├── 快速启动.bat            # Windows 一键启动 ⭐
├── 快速构建.bat            # 构建脚本
└── 打包分享.bat            # 打包脚本（你正在用的）
```

## 💡 给接收者的建议

### 前置要求

对方的电脑需要：
- Windows/Mac/Linux 系统
- 安装了 Node.js（https://nodejs.org/）

### 如果对方不想安装 Node.js

推荐部署到云服务（方式二），这样对方只需要访问网址即可，不需要安装任何东西！
