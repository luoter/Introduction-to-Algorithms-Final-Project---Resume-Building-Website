# 🚀 超级快速开始指南

## 方法一：Windows 用户一键启动（最简单！）

如果你是 Windows 用户，只需要两步：

1. **双击运行 `快速启动.bat`**
2. **等待浏览器自动打开**

就这么简单！🎉

---

## 方法二：Mac/Linux 用户

打开终端，进入项目文件夹，然后运行：

```bash
# 安装依赖（第一次运行需要）
npm install

# 启动服务器
npm run dev
```

然后在浏览器中打开显示的地址。

---

## 方法三：直接使用在线版本

如果你不想在本地安装任何东西，可以：

1. 部署到 Vercel（免费）
2. 部署到 Netlify（免费）
3. 部署到 GitHub Pages（免费）

详见 [README.md](README.md) 中的详细说明。

## ✨ 使用技巧

### 💾 数据保存
你的所有数据会自动保存在浏览器本地，刷新页面不会丢失！

### 📄 导出 Word
编辑完成后，点击右上角「📄 导出 Word」即可下载专业简历文档。

### 🤖 AI 优化
每个模块都有「✨ AI 优化」按钮，点击后会智能优化你的内容描述。
