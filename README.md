# 快速启动指南

## Windows 推荐方式

1. 安装 Node.js。
2. 打开项目文件夹。
3. 双击 `快速启动.bat`。
4. 浏览器访问终端显示的地址，例如：

```text
http://localhost:3000/
```

如果 3000 端口被占用，地址可能会变成：

```text
http://localhost:3001/
```


## 使用说明

- 直接填写左侧表单，右侧会实时预览简历。
- 使用“模块排序推荐”下拉框，可以根据求职场景调整模块顺序。
- 点击“保存简历”会把数据保存到浏览器本地。
- 点击“导出 Word”可以下载简历文档。
- 点击“导出网站”可以下载个人展示网页。
- `AI 生成`、`AI 优化` 需要先在“设置”里配置 API Key。

## API Key

支持两种 AI 提供商：

- Kimi：https://platform.moonshot.cn
- 硅基流动：https://www.siliconflow.cn

配置步骤：

1. 在对应平台注册账号。
2. 创建 API Key。
3. 回到项目页面，点击“设置”。
4. 选择 AI 提供商。
5. 粘贴 API Key 并保存。

## 注意事项

- API Key 不要截图或发给别人。
- 本地使用可以把 API Key 存在浏览器里。
- 如果部署到公网，建议改成后端调用 AI API。
- 如果启动时报 `.vs` 文件夹相关错误，可以关闭 Visual Studio 后删除或重命名 `.vs` 文件夹。
